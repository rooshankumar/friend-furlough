Multiple caches can cause old code to persist:

1. VITE CACHE
   Location: node_modules/.vite/
   Problem: Stores old build artifacts

2. ANDROID BUILD CACHE
   Location: android/app/build/, android/build/, android/.gradle/
   Problem: Old compiled APK files

3. ANDROID WEB ASSETS
   Location: android/app/src/main/assets/public/
   Problem: Old web files from previous sync

4. ANDROID STUDIO CACHE
   Location: System cache (varies by OS)
   Problem: IDE caches old project state

5. DEVICE CACHE
   Problem: Old APK installed on device
